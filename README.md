# proyecto_s7
Proyecto del Sprint 7 - Análisis de ventas de vehículos

## Descripción del proyecto
Esta aplicación web permite a los usuarios explorar interactivamente las ventas de diferentes vechículos así como los datos de los anuncios.

## Funcionalidad
* Visualización de histogramas para analizar la distribución de precios, modelos, etc.
* Filtros interactivos mediante casillas.
* Análisis exploratorio de datos en tiempo real.

## Paquetes utilizads
Python
Pandas
Ploty express
Streamlit
